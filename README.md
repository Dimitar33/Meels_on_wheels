
# Production MERN Secure Notes App

## Features
- JWT Access + Refresh Authentication
- MongoDB Database
- Tailwind UI with Dark Mode
- Dockerized Setup
- Secure Express Backend

## Run with Docker
docker-compose up --build

Frontend: http://localhost:5173
Backend: http://localhost:5000

## Manual Run
Server:
cd server
npm install
npm start

Client:
cd client
npm install
npm run dev
